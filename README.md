# Setup:

Create a `secrets.tfvars` file in the `opentofu` directory with these values:
```
DISCORD_PUBLIC_KEY = ""
DB_HOST = ""
DB_PORT = 
DB_NAME = ""
DB_USER = ""
DB_PASSWORD = ""
```

# Deploy:

### MacOS:

Run the `deploy.sh` script in the root folder

### Windows:

Run the `deploy.ps1` script in the root folder
